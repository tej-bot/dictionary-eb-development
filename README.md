# Dictionary Website

## App Overview / Features
This is a dictionary web application built using HTML, CSS, and JavaScript.
It allows users to search for words and view their meanings in a simple and clean interface.

Features:
- Search for word meanings
- User-friendly interface
- Fast and lightweight
- Runs directly in the browser

2. Tech Stack Used
- HTML
- CSS
- JavaScript

## Steps to Run Locally

1. Clone the repository
   git clone https://github.com/tej-bot/dictionary-eb-development.git

2. Navigate to the project folder
   cd dictionary-eb-development

3. Run the project
   Open the `index.html` file in any web browser
